# YOLO Smart Vision System

A computer vision analytics platform based on YOLOv8, integrating object detection, multi-object tracking, heatmap visualization, ROI analysis, and behavioral statistics.

---

## Project Overview

YOLO Smart Vision System is designed for intelligent video analysis scenarios such as crowd monitoring, traffic analysis, campus surveillance, and behavioral understanding.

The system combines real-time object detection with trajectory analysis and heatmap visualization to provide intuitive insights into object distribution, movement patterns, and regional activity.

---

## Key Features

### Object Detection

* YOLOv8-based real-time object detection
* Multi-class target recognition
* Support for images, videos, and webcam streams

### Multi-Object Tracking

* Continuous target identity tracking
* Trajectory visualization
* Target counting and movement analysis

### ROI Analysis

* Interactive ROI selection
* Region-based activity statistics
* Entry and exit counting

### Heatmap Visualization

* Real-time heatmap generation
* Heatmap overlay on original video
* Activity density visualization
* Temporal heatmap analysis

### Analytics Module

* Behavioral statistics
* Crowd density estimation
* CSV data export
* Motion trend analysis
